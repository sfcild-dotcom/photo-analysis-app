# 띄부띄부 사진 분석기 (Photo Analysis App)

이 앱은 구글의 Gemini AI를 활용하여 업로드된 사진을 정밀하게 분석하고, 그 결과를 바탕으로 전문적인 블로그 원고 초안을 생성하는 도구입니다.

## 🚀 주요 기능
- **사진 다중 분석**: 최대 14장의 사진을 한 번에 업로드하여 각 사진의 시각적 요소(색감, 사물, 분위기)를 분석합니다.
- **AI 기반 리포트**: Gemini 1.5 Pro(또는 Flash) 엔진을 통해 각 문단에 필요한 현장감 있는 묘사를 텍스트로 변환합니다.
- **Vercel 배포**: 빠르고 안정적인 클라우드 환경에서 동작합니다.

## 🛠️ 설치 및 설정
1. **API 키 설정**: Vercel의 `Environment Variables`에 `VITE_GEMINI_API_KEY`라는 이름으로 키를 등록하세요.
2. **배포**: GitHub 저장소와 연결된 Vercel 프로젝트에서 `Redeploy`를 실행합니다.
